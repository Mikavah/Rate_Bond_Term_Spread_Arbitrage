# RateBondTermSpreadArbitrage
## 项目介绍
本项目聚焦利率债核心套利策略——10年期与1年期国债利差期限套利，基于**均值回归理论**构建量化交易模型，涵盖数据获取（Wind/公开数据源）、策略逻辑实现、回测分析全流程，完全贴合固收量化交易岗位的实际工作场景。策略具备参数可调整、风险可控、回测指标完善的特点，可直接用于账户业绩管理、交易策略优化及内部系统开发参考。

## 策略核心逻辑
利率债的期限利差（10Y-1Y国债收益率）存在显著的均值回归特性：当利差偏离历史合理区间时，市场力量会推动利差回归至均值水平。基于此构建套利策略：
1. 利差上行突破上轨（均值+2σ）：做空利差（做多1Y国债，做空10Y国债）
2. 利差下行突破下轨（均值-2σ）：做多利差（做空1Y国债，做多10Y国债）
3. 利差回归至均值区间：平仓止盈
4. 加入止损/仓位控制：控制单笔风险与总仓位，符合机构风控要求

## 数学模型
### 1. 核心指标计算
#### （1）期限利差
$$spread_t = Y_{10Y,t} - Y_{1Y,t}$$
其中：
- $Y_{10Y,t}$：t时刻10年期国债收益率（中债国债到期收益率：10年）
- $Y_{1Y,t}$：t时刻1年期国债收益率（中债国债到期收益率：1年）

#### （2）滚动均值与标准差（布林带）
采用滚动窗口计算利差的统计特征：
$$\mu_t = \frac{1}{N}\sum_{i=t-N+1}^{t} spread_i$$
$$\sigma_t = \sqrt{\frac{1}{N-1}\sum_{i=t-N+1}^{t} (spread_i - \mu_t)^2}$$
其中：N为滚动窗口长度（默认60个交易日）

#### （3）交易信号阈值
上轨：$upper\_band_t = \mu_t + k \times \sigma_t$（k默认2）
下轨：$lower\_band_t = \mu_t - k \times \sigma_t$（k默认2）
中性区间：$[\mu_t - 0.5\sigma_t, \mu_t + 0.5\sigma_t]$

### 2. 交易信号生成
| 利差状态                | 交易信号       | 具体操作                          |
|-------------------------|----------------|-----------------------------------|
| $spread_t > upper\_band_t$ | 做空利差       | 做多1Y国债，做空10Y国债           |
| $spread_t < lower\_band_t$ | 做多利差       | 做空1Y国债，做多10Y国债           |
| 利差回归中性区间        | 平仓止盈       | 平掉当前所有头寸                  |
| 单笔亏损≥止损阈值       | 止损平仓       | 平掉当前所有头寸                  |

### 3. 风险控制模型
- 单笔最大亏损：账户净值的0.5%（可配置）
- 总仓位限制：不超过账户总资金的50%（可配置）
- 最大持仓周期：单次持仓不超过20个交易日（防止极端行情）
